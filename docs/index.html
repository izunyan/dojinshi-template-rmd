テンプレートの説明

  • 同人誌の原稿を書くためのテンプレート
  • Build Bookすればhtmとpdfがそのままできる
    • htmlはbs4_bookスタイル
    • pdfは目次から始まり，ページ番号が1から始まる

実行方法

  • Buildタブで
    • Build Book > bookdown::bs4_book
    • Build Book > bookdown::pdf_book
  • Knitでやるとエラーになるよ！
